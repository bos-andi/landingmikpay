# Gallery Images untuk MikPay

Folder ini berisi screenshot/gambar untuk halaman gallery MikPay.

## Nama File yang Diperlukan:

1. **dashboard-overview.png**
   - Deskripsi: Screenshot dashboard utama MikPay
   - Menampilkan: Overview aktivitas, statistik, dan monitoring real-time

2. **hotspot-payment.png**
   - Deskripsi: Screenshot halaman manajemen pembayaran hotspot
   - Menampilkan: Daftar pengguna hotspot dengan status pembayaran

3. **payment-billing-system.png**
   - Deskripsi: Screenshot sistem billing dan tagihan
   - Menampilkan: Interface generate tagihan, tracking pembayaran

4. **voucher-payment.png**
   - Deskripsi: Screenshot sistem voucher dengan payment
   - Menampilkan: Generate voucher, quick print, tracking penjualan

5. **payment-reports.png**
   - Deskripsi: Screenshot laporan pembayaran dan keuangan
   - Menampilkan: Grafik pendapatan, laporan harian/bulanan, export data

6. **ppp-billing.png**
   - Deskripsi: Screenshot sistem billing PPP
   - Menampilkan: Manajemen PPP secrets dengan tracking pembayaran

## Spesifikasi Gambar:
- Format: PNG atau JPG
- Ukuran rekomendasi: 1200x675px (16:9 aspect ratio)
- Kualitas: High resolution untuk tampilan yang jelas
- File size: Maksimal 500KB per gambar untuk optimasi loading

## Catatan:
Jika gambar belum tersedia, halaman akan menampilkan placeholder dengan icon yang sesuai.
